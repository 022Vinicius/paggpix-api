# paggpix-api
essa API é uma solução de backend desenvolvida em NestJS para o Teste Técnico Paggpix. Ela é responsável por gerenciar a criação, listagem, busca e atualização de pagamentos, além de implementar uma camada de autenticação baseada em token (client_id_token).
